# Working Notes Index

> **Назначение**: Рабочие заметки для проектирования, обсуждений и предварительной аналитики до фиксации проблемы или решения в ADR.

---

## 📋 Правила использования

- Рабочая заметка — место для исследования, сравнения вариантов и промежуточных выводов.
- Финальные архитектурные проблемы фиксируются в `docs/adr/*/*-PROBLEM-*.md`.
- Финальные архитектурные решения фиксируются в `docs/adr/*/*-DEC-*.md`.
- Заметки не заменяют ADR и не являются источником окончательных проектных решений.
