# connector/domain/dsl/loader

## Назначение

Утилиты загрузки YAML-артефактов для DSL. Содержит общую логику чтения файлов реестра и базовые хелперы парсинга, переиспользуемые всеми DSL-лоадерами.

## Файлы

| Файл | Назначение |
|---|---|
| `_common.py` | `load_registry(path)` — читает `registry.yaml`; базовые YAML-хелперы, используемые другими лоадерами |

## Зависимости

**Зависит от:** `PyYAML`, stdlib.  
**Используется:** `datasets/registry.py`, DSL-лоадерами во всём `domain/`.
